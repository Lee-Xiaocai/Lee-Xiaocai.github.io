PHP部分

新建一个名字为coon.inc.php的文件。这个文件主要配置连接数据库的信息。我的如下：

<?php
define("HOST",'192.158.227.156');
define("USER",'你数据库用户名');
define("PWD",'你的数据库密码');
?>

新建一个名字为coon.php的文件，用于连接数据库。我的如下：

<?php
include "conn.inc.php";
$link = mysqli_connect(HOST, USER, PWD);
if ($link->connect_erro) {
    die('数据库链接出错');
}

$result = mysqli_select_db($link, "你的数据库名称");

if (!$result) {
    die("没有选择任何数据库");
}
?>

数据库部份

在数据库中新建一个表名为Api_list的数据表。设置如图：

通过以下Sql语句添加一个Api：

"INSERT INTO `Api_list`(`ID`, `Api_name`, `Api_src`) VALUES ('','Api名称','Api地址')"

想要删除直接点击删除操作或者
通过以下Sql语句删除一个Api：

"DELETE FROM Api_list WHERE id='要删除APi的ID'"
